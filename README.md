# Amazon Kindle 実質割引率チェッカー (Amazon Kindle Real Discount Rate Checker)

Amazon.co.jpのKindleストアにおいて、ポイント還元やクーポン割引を考慮した **実質割引率** を自動計算し、お得な書籍を視覚的にハイライトするChrome拡張機能です。

実質割引率の表示基準（15%カットオフ）や赤系カラー設定を用いて、視認性と美しさを重視して設計されています。

> [!WARNING]
> **免責事項**
> 表示される価格/割引/還元等の情報には誤りが含まれることがあります。
> 購入前に必ず公式情報をご確認ください。購入は自己責任でお願いいたします。

---

## 主な機能

1. **実質価格・実質割引率の自動計算**
   - 「紙の本の価格（または元価格）」を基準に、現在の「Kindle価格」「獲得ポイント」「割引クーポン（%割引または円引き）」を自動で解析し、実質的な価格と割引率を計算します。
2. **スマートなバッジ表示 (15%しきい値)**
   - 実質割引率が **15%以上** の商品に対してのみ、カードの左上に「お得」「注目」「大特価」「超特価」「神割引」といったバッジを表示します。
   - 15%未満の割引率が低い商品はバッジを表示せず、画面をスッキリと保ちます。
3. **ホバー詳細ツールチップ**
   - 割引バッジにマウスを乗せると、基準価格、Kindle価格、ポイント還元額、クーポン割引額、最終的な実質価格の内訳がツールチップで一覧表示されます。
4. **段階的な赤系ハイライト（ピンク〜赤のグラデーション）**
   - 実質割引率に応じて、カードの枠線（ボーダー）と背景色を薄いピンクから鮮やかな赤へと段階的に強調表示します。
   - 背景色は割引率の高さに応じて動的に不透明度（アルファ値 3%〜18%）が調整されます。
5. **ポップアップでの凡例確認**
   - 拡張機能アイコンをクリックすると、割引率のしきい値と色分けの凡例をいつでも確認できます。
6. **動的読み込み・Shadow DOM対応**
   - 無限スクロールやタブ切り替え、React等の非同期描画によるコンテンツ追加も監視し、自動で割引計算を適用します。また、Amazon独自のShadow DOM内部にある書籍情報も漏れなくスキャンします。

## 割引率段階とデザイン一覧

| 割引率の範囲 | 段階名 | カードの枠線・背景色 | バッジデザイン |
| :--- | :--- | :--- | :--- |
| **90% 以上** | **神割引！** | 太い赤の破線 / 濃い赤背景 (18%) | 赤枠線・赤テキスト (パルスアニメーション) |
| **70% 〜 90% 未満** | **超特価** | 濃いピンクレッドの実線 / 赤背景 (14%) | 濃いピンクレッド枠線・テキスト |
| **50% 〜 70% 未満** | **大特価** | ディープピンクの実線 / 赤背景 (10%) | ディープピンク枠線・テキスト |
| **30% 〜 50% 未満** | **注目** | ローズピンクの実線 / 赤背景 (6%) | ローズピンク枠線・テキスト |
| **15% 〜 30% 未満** | **お得** | 薄いピンクの実線 / 赤背景 (3%) | 薄いピンク枠線・テキスト |
| **15% 未満** | (非表示) | 装飾なし (Amazon標準のまま) | バッジ非表示 |

## インストール方法 (デベロッパーモード)

現在、本拡張機能はパッケージ化前の開発版として提供されています。以下の手順でインストールできます。

1. このリポジトリをクローンするか、ZIPファイルとしてダウンロードして展開します。
2. Google Chrome を起動し、アドレスバーに `chrome://extensions/` と入力して拡張機能ページを開きます。
3. 画面右上にある **「デベロッパーモード」** のスイッチをオンにします。
4. 画面左上の **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5. 展開したプロジェクトのフォルダ（`manifest.json` が含まれているディレクトリ）を選択して読み込みます。

## 動作対象ページ

Amazon.co.jpにおける以下のKindle書籍一覧ページが対象となります。
- 「続きの本・シリーズの関連本」などの特設セール一覧 (`/seeMore` などのパスを含むURL)
- シリーズセールの詳細ページ等

---

## ファイル構成

- `manifest.json` : 拡張機能のメタデータおよび設定ファイル (Manifest V3 準拠)。
- `content.js` : Amazonの商品一覧ページで価格情報を抽出し、バッジやスタイルを適用するメインスクリプト。
- `content.css` : 商品カードや割引バッジ、詳細ツールチップ用のスタイルシート。
- `popup.html` / `popup.css` : ツールバーのアイコンをクリックしたときに表示される凡例UI。
- `icons/` : 拡張機能用のアイコン画像群。

## スペシャルサンクス
